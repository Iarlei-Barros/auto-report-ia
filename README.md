# Auto Report com IA (RPA)

## Sobre o projeto

Sistema de automação (RPA) desenvolvido em Node.js que coleta dados automaticamente de uma API, processa as informações e gera relatórios com insights, simulando o uso de Inteligência Artificial.

## Problema

Empresas frequentemente lidam com tarefas repetitivas e grandes volumes de dados, o que consome tempo, reduz produtividade e aumenta a chance de erros manuais.

## Solução

Este projeto automatiza:

* Coleta de dados via API
* Processamento e organização das informações
* Geração de relatórios automatizados

## Funcionalidades

* Integração com API externa
* Processamento de dados
* Geração de relatórios (`.txt` e `.json`)
* Logs de execução
* Tratamento de erros

## Tecnologias

* Node.js
* JavaScript
* Axios

## Estrutura do projeto

src/
├── services/
├── utils/
└── index.js

## Como executar

```bash
npm install
npm start
```

## Objetivo

Demonstrar habilidades em automação de processos, manipulação de dados e organização de código.
